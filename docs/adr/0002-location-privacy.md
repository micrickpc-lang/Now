# ADR-0002: уровни и жизненный цикл геолокации

Статус: принято.

Сигнал хранит только выбранный уровень раскрытия. `none`, `city` и `district` не требуют координат; `approximate` записывает только необратимо огрублённую точку (исходная точка не сохраняется); `exact_room` хранится отдельно, зашифрована envelope encryption и физически удаляется при revoke, leave, block, room completion или TTL. Доступ разрешён только активному room member после block check; чтение аудитируется. Background permission приложению не запрашивается.
