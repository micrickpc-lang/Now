# Модерация

Пользователь может пожаловаться на одного субъекта: пользователя, сигнал или сообщение. Категории: spam, fraud, threats, harassment, doxxing, unsafe meeting, content, impersonation, other. Быстрая block operation сразу удаляет дружбу, ограничивает совместную room membership и отзывает взаимно доступные exact locations.

Admin queue закрыта RBAC-сессией. Решения `investigate`, `warn`, `suspend`, `dismiss`, `restore` требуют текстовую причину. Suspend отзывает сессии и location shares. Каждое действие записывает admin id, report id, reason и время.

Production SLA и escalation:

- угроза жизни/опасная встреча/данные несовершеннолетнего — немедленная приоритизация и утверждённый emergency runbook;
- doxxing/threats — ограничение видимости до решения;
- spam — стандартная очередь;
- апелляция рассматривается другим модератором, когда команда позволяет разделение обязанностей.

Просмотр текста сигнала/сообщения администратором допустим только внутри конкретного расследования. Массовый browsing закрытого контента запрещён организационной политикой и техническим audit.
